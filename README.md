## 2. Дерево решений (Decision Tree)

Дерево решений – это бинарное дерево, где каждый внутренний узел проверяет значение одного признака и сравнивает его с порогом. В зависимости от результата пример направляется в левую или правую ветвь. Лист дерева содержит окончательное предсказание (класс). Дерево строится рекурсивно, на каждом шаге выбирая такое разбиение, которое максимально «очищает» данные (уменьшает примесь Джини).

## 3. Бэггинг (Bagging)

Беггинг- обучает много деревьев на разных случайных подвыборках исходных данных и усредняет их предсказания ,что понижает переобучение и повышает точность

# Bootstrap aggregating
Обучается множество разных деревьев. Каждое дерево получает свою обучающую выборку, сгенерированную рандомно. Размер выборки совпадает с размером исходной.

# Обучение ансамбля
- Для каждого дерева создаётся bootstrap‑выборка (`bootstrap_sample`).
- Дерево обучается на этой выборке (`fit_tree`).
- Все деревья сохраняются в массиве `trees` структуры `BegModel`.

# Предсказание
При предсказании опрашиваются все деревья. Каждое дерево говорит за класс, который оно предсказало. Итоговым ответом становится класс, набравший **максимальное количество голосов** (функция `max_pred`).
